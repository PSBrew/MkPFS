# Research: Update Reports

Generate a paired markdown + HTML report for the latest phase.

## Prompt Template

```
Generating phase report. What should we title this phase?

Detected findings since last report:
- <auto-detected from recent KB docs and scripts>

Suggested title: "<inferred title>"
Accept or rename?
```

## Report Numbering

Scan `reports/` for existing files. Next number = max(existing NNN) + 1, zero-padded to 3 digits.

## Markdown Template (`reports/NNN_title.md`)

```markdown
# Phase N Report: <Title>

**Date:** <YYYY-MM-DD> · **Phase:** <Phase Name>

## Investigation Scope
<What was investigated and why>

## Findings
| Component | Status | Notes |
|-----------|--------|-------|
| <item> | Confirmed/Blocked/Partial | <detail> |

## Key Discoveries
### <Discovery 1 title>
<Description with evidence references>

### <Discovery 2 title>
<Description with evidence references>

## Artifacts Produced
- `scripts/NNNN_*.py` — <description>
- `knowledge_base/NNN_*.md` — <description>

## Next Phase
1. <Next step 1>
2. <Next step 2>
```

## HTML Template (`reports/NNN_title.html`)

Use the dark-themed report template. Key style variables:
- Background: `#0b1120`
- Surface: `#111827`
- Card: `#1e293b`
- Blue accent: `#3b82f6`
- Green (confirmed): `#22c55e`
- Red (blocked): `#ef4444`
- Orange (partial): `#f59e0b`

Structure:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Phase N Report: <Title></title>
  <style>/* dark theme CSS from existing report template */</style>
</head>
<body>
  <a href="../STATUS.html" class="back-link">← Back to Dashboard</a>
  <h1>Phase N Report: <Title></h1>
  <div class="meta">
    <span class="item">📅 <date></span>
    <span class="item">📍 Phase: <name></span>
    <span class="item">📊 Progress: <pct>%</span>
  </div>

  <h2>🔍 Investigation Scope</h2>
  <p><scope></p>

  <h2>📊 Findings</h2>
  <table><!-- status badges with status-ok/status-blocked/status-partial --></table>

  <h2>💡 Key Discoveries</h2>
  <!-- .finding cards with border-left accent -->

  <h2>📄 Artifacts Produced</h2>
  <ul><!-- script and KB links --></ul>

  <h2>📍 Next Phase</h2>
  <ol><!-- next steps --></ol>

  <div class="footer">
    Generated by <code>research-update-reports</code> · <a href="../STATUS.html">Dashboard</a>
  </div>
</body>
</html>
```

Status badge classes:
- `status-ok` (green bg) for Confirmed
- `status-blocked` (red bg) for Blocked
- `status-partial` (orange bg) for Partial

## Steps

1. Determine next report number by scanning `reports/`
2. Ask user for phase title (or infer from current work)
3. Summarize findings from scripts and knowledge_base entries since last report
4. Create `reports/NNN_<phase_title>.md` using markdown template
5. Create `reports/NNN_<phase_title>.html` using dark-themed HTML template
6. Update INDEX.md with new report entries
7. Update STATUS-data.json:
   - Bump `stats.reports` count
   - Add new report to `key_documents[]` with icon `📰`
   - Set `meta.generator` to `research-update-reports`
